# 📌 Sobre o Projeto

API REST desenvolvida utilizando Node.js, Express e MySQL, responsável por retornar uma listagem de médicos para consumo em aplicações Front-end.

O projeto foi criado com foco em praticar conceitos de desenvolvimento Back-end, integração com banco de dados e construção de APIs modernas.

---

# 🚀 Tecnologias Utilizadas

- Node.js
- Express
- MySQL
- JavaScript
- API REST
- dotenv
- Git & GitHub

---

# ✨ Funcionalidades

✅ Conexão com banco de dados MySQL  
✅ Retorno de dados em JSON  
✅ Estruturação de rotas REST  
✅ Organização de código  
✅ Integração com aplicações Front-end  
✅ Hospedagem da API na nuvem  

---

# 📂 Estrutura da API

```bash
GET /medicos
```

Retorna uma lista de médicos cadastrados no banco de dados.

# ⚙️ Como Executar o Projeto

## Clone o repositório

```bash
git clone https://github.com/GabrielFR-Dev/APIs-Listagem-Contato
```

---

## Acesse a pasta

```bash
cd APIs-Listagem-Contato
```

---

## Instale as dependências

```bash
npm install
```

---

## Configure o arquivo `.env`

```env
CL_HOST=
CL_USER=
CL_PASSWORD=
CL_DATABASE=
```

---

## Execute o projeto

```bash
npm start
```
